# 가장 어려운 traits다.

- traits는 누구나 어렵다. 유일한 방법은 수많은 예시를 해보고, 울면서 cargo에 거부 당하면서, 고통 받다보면, 알아서 실력이 향상되는 유일한 방법이다. 무식하지만 확실하다.

- 지금까지 상단에 아무 생각없이 쓰던 derive의 정체는 trait다. 기본적인 트레이트는 암기하도록하자.
  - Debug, Default, PartialEq, Eq 등등

```rs
#[derive(Debug)]

```
